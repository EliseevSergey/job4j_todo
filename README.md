**job4j_todo**

Pet project on job4j course.

ToDo list application with pages for:<br>
-Registration/Login<br>
-Add tasks and set statuses<br>
-Set completion status<br>
-Edit tasks<br>
-Filtering tasks with completion status<br>


**Stack:**
Java 17, SpringBoot 2.7.6, Thymeleaf, Bootstrap,
Maven 4.0.0, Liquibase 4.15.0, Hibernet 5.6.11
PostgreSQL 42.2.9

**Requirements:**
Java 17, SpringBoot 2.7.6, Thymeleaf, Bootstrap,
Maven 4.0.0, Liquibase 4.15.0, Hibernet 5.6.11
PostgreSQL 42.2.9

**Installation**<br>
-Create database todo<br>
-Maven Install <br>
-Run Main class<br>
